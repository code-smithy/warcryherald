# English And German Example Dataset

This is fictional data that demonstrates how to model multiple languages with
the current Phase 3 schema. Do not import it as real Warcry data.

Until dedicated translation tables exist, each language is represented as its
own source document, rules release, and language-suffixed stable-key slice.
Future languages should follow the same pattern, for example `fr`, `es`, `it`,
`ja`, or `ko` suffixes.
